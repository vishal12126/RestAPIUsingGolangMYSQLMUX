It needs to be setup so that Code is found in your PATH. If you're on a mac system, do the following (for windows systems, read below):

Launch VS Code
Open up command palette (press F1) and type shell command to find Shell Command: Install 'code' command in PATH command.
Restart terminal

If you're on Windows, you can also set the PATH manually by adding:

C:\Program Files (x86)\Microsoft VS Code\bin
to your PATH environment variable.

Open command prompt
type setx path "%path%;C:\Program Files (x86)\Microsoft VS Code\bin"
Restart terminal
